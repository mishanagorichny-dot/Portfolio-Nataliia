# Nataliia — UX/UI Designer Portfolio

Personal portfolio site for Nataliia, Junior UX/UI Designer.

## 🚀 Deploy to GitHub Pages

1. Create a new repository on GitHub (e.g. `nataliia-portfolio`)
2. Push this folder:
```bash
git init
git add .
git commit -m "Initial portfolio"
git remote add origin https://github.com/YOUR_USERNAME/nataliia-portfolio.git
git push -u origin main
```
3. Go to **Settings → Pages → Source: Deploy from branch → main / root**
4. Your site will be live at: `https://YOUR_USERNAME.github.io/nataliia-portfolio`

## 📝 What to update before publishing

In `index.html`, replace these placeholders:

| Placeholder | What to put |
|---|---|
| `your@email.com` | Your actual email |
| `https://linkedin.com/in/` | Your LinkedIn URL |
| `https://behance.net/` | Your Behance URL |
| `https://t.me/` | Your Telegram username |

## 🛠 Tech Stack

- Pure HTML + CSS + Vanilla JS (no build step needed)
- Google Fonts: DM Serif Display + Inter
- Scroll reveal animations
- Fully responsive (mobile, tablet, desktop)
- Images embedded as base64 (no server needed)
